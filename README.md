# Book-App

A book exchange and resale platform in which users can exchange their old books or sell

## Screenshots




## Features

- Old book exchange with other user
- Old book resale
- Similar Book Recommandation using AI


## Technical Overview

- Spring Security
- Spring Data Jpa

- JWT
- Deploy to cloud VPS server
